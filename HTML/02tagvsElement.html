Tag vs Element

Tags

Tags Are come in between < angularbrackets>. like h1, h2, p, head, body etc

90% tags are come with opening/ Start tag and closing tags/ End tag.

10% tags are self closing tags, i.e br, hr, img etc, these are also called void tag / empty tag / self closing tag

Elements

Start tag - some content - End tag

Attributes

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.
